##  Features

- **Welcome**, **Games**, **Writing**, and **About** pages  
- Organized presentation of creative work and personal projects  
- Responsive layout using standard HTML/CSS  
- Flask backend serving Jinja2 templates

---

##  Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML5, CSS3, Jinja2  
- **Deployment:** Render (via Gunicorn)
